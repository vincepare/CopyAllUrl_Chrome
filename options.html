Custom format

Set your own row format by writing the template in this textarea.
Use $title and $url variables.
Don't forget to add a new line at the end, instead you'll get all copied data into a single line !

Miscellaneous

Extract URLs from raw text when using paste feature (instead of processing pasted text line by line, with 1 line = 1 url). The only pitfall using this feature is that it doesn't get "special urls" like ones beginning with file:/// or chrome://...
Copy only selected tabs (you can select multiple tabs by holding the Ctrl key when clicking a tab).
Grab tabs URL from all chrome windows, not only the current one.
Choose what to do when clicking the umbrella icon. Specially designed for lazy people who use only one feature (copy or paste) and want to fire this feature directly with one click, without showing the drop down menu...
Use HTML MIME type to copy data as HTML rather than plain text, allowing to embed url into clickable links when pasting on word or an email client. This only affects HTML and Custom formats.

How to use ?

Simply click the umbrella icon, and choose Copy or Paste !

You can also define keyboard shorcut for copy and paste feature on chrome extensions page (chrome://extensions/).

FAQ

Can CopyAllURLs copy tab history ?

Unfortunatly, since people at Google Chrome & Chromium projects don't want to give access to tab history via Chrome API, it is not possible to do this -_-'.

You can read this topic for more information : http://code.google.com/p/chromium/issues/detail?id=41321.


Can I use keyboard shortcuts (hotkeys) ?

Yes you can, go to chrome extensions management (chrome://extensions/), click "Configure commands" at the bottom right to show a popin where you can customize shortcuts for extensions that handles it (including Copy All Urls). More here and there.
